# Music Lesson App

This app lets teachers create personalized profiles showcasing their expertise, instruments, and teaching styles.   
Alternatively, students can easily search and discover the perfect instructor based on their musical preferences, skill levels, and learning goals.    

# Features

* Register as a professional music teacher - create, update and delete profile
* featuring an 'away' and 'no longer taking students' option for teachers
* Filter through prospective teachers  
* Schedule lessons - date, time, duration
* Chat box to talk directly with teachers
* Student can set up automatic payments - teachers will be able to recieve their payments
